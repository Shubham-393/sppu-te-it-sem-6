Tennis Rules

  • Matches can be best of 3 or 5 sets.
  • Points: Love, 15, 30, 40, and Game.
  • A player must win a set by 2 games.
  • Serve alternates after every game.
  • Tiebreak is used if a set reaches 6–6.
